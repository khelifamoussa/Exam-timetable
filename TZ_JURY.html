How to study each exercise

Each exercise is now written like an exam problem: first understand the block diagram, then build the transfer function, then derive the reference equation, then compute numerical values, and finally decide stability.

كل تمرين مكتوب الآن مثل مسألة امتحان: نفهم أولاً المخطط الكتلي، ثم نبني دالة التحويل، ثم نستخرج المعادلة المرجعية، ثم نحسب القيم العددية، وأخيراً نحكم على الاستقرار.

General closed-loop rule

\[F(z)=\frac{Y(z)}{R(z)}=\frac{L(z)}{1+L(z)},\qquad L(z)=C(z)G(z)H(z)\]
دالة التحويل المغلقة تساوي دالة الحلقة المفتوحة مقسومة على واحد زائد دالة الحلقة المفتوحة.

Stability principle

\[|z_i|<1\quad \text{for every closed-loop pole}\]
النظام مستقر إذا كانت جميع أقطابه داخل الدائرة الواحدة.

Universal solution algorithm

Draw or read the block diagram

Identify reference, comparator, controller, controlled dynamics, feedback and output.

Find \(F(z)\)

Apply the negative-feedback formula and simplify.

Extract \(D(z)\)

The denominator is the characteristic polynomial.

Build the recurrence equation

Divide by the highest power of \(z\), then apply inverse Z-transform.

Compute step response

Use \(r[n]=1\) and zero initial conditions.

Check stability and error

Use Jury or roots, then compute \(e_\infty=1-F(1)\).

Jury Stability Method — Complete Academic Explanation

Purpose

The Jury method decides whether all roots of a discrete-time characteristic polynomial are inside the unit circle. This is essential because a stable discrete-time system must have all poles strictly inside \(|z|=1\).

تهدف طريقة جيري إلى معرفة هل جذور كثيرة الحدود المميزة كلها داخل الدائرة الواحدة، لأن استقرار النظام المتقطع يتطلب أن تكون كل الأقطاب داخل \(|z|=1\).

Polynomial form

\[D(z)=a_0z^n+a_1z^{n-1}+a_2z^{n-2}+\cdots+a_n\]
نكتب كثيرة الحدود بترتيب تنازلي للدرجات.

Preliminary conditions

\[D(1)>0,\qquad (-1)^nD(-1)>0,\qquad |a_n|
إذا فشل أحد هذه الشروط يكون النظام غير مستقر. وإذا نجحت نكمل جدول جيري.

Full Jury table construction

General construction stages

Original row

\[R_1=[a_0,\ a_1,\ldots,\ a_n]\]

Reversed row

\[R_2=[a_n,\ a_{n-1},\ldots,\ a_0]\]

Reduction row

\[q_k=p_0p_k-p_mp_{m-k}\]

The last term becomes zero and is removed. Repeat until reaching the final low-order stage.

Decision

After reaching the final stage, apply the remaining stability conditions. If all stages pass, the system is stable.

Exercise 1 — Unity feedback system \(G(z)=\frac{2z}{z-0.5}\)

Q1Draw the block diagram and identify the signals

1 Block diagram — Simple unity-feedback dynamic system
المرجع \(R(z)\) يقارن مع الخرج \(Y(z)\)، والفرق يدخل إلى النظام الديناميكي. التغذية الراجعة موحدة وسالبة.

\[F(z)=\frac{G(z)}{1+G(z)}=\frac{\frac{2z}{z-0.5}}{1+\frac{2z}{z-0.5}}=\frac{2z}{3z-0.5}\]

\[D(z)=3z-0.5\]

Exercise 2 — Dynamic system with a digital integrator

Q1Draw the block diagram

2 Block diagram — Digital integrator before the controlled dynamic system
هنا أضفنا مكاملاً رقمياً قبل النظام المتحكم فيه. وجود المكامل يراكم الخطأ ويحسن الدقة الساكنة، لكنه يرفع درجة النظام لذلك يجب التحقق من الاستقرار.

Q2Find the open-loop and closed-loop functions

\[L(z)=C(z)G(z)=\frac{1}{z-1}\cdot \frac{2z}{z-0.5}=\frac{2z}{(z-1)(z-0.5)}\]
\[F(z)=\frac{L(z)}{1+L(z)}=\frac{2z}{(z-1)(z-0.5)+2z}\]

Q3Find the characteristic polynomial

\[(z-1)(z-0.5)=z^2-1.5z+0.5\]
\[D(z)=z^2-1.5z+0.5+2z=z^2+0.5z+0.5\]

Q4Verify stability using Jury

\[D(1)=1+0.5+0.5=2>0\]
\[D(-1)=1-0.5+0.5=1>0\]
\[|0.5|<1\]
The system is stable.

Q5Derive recurrence equation and explain the effect of the integrator

\[(z^2+0.5z+0.5)Y(z)=2zR(z)\]
\[y[n]+0.5y[n-1]+0.5y[n-2]=2r[n-1]\]
\[y[n]=-0.5y[n-1]-0.5y[n-2]+2r[n-1]\]
The integrator improves tracking accuracy, but it also increases the order of the system. Therefore, stability must always be checked after adding it.

Expanded Drone Exercise — Five Questions with Detailed Solution

Engineering context

A drone altitude-control system must force the measured altitude \(y[n]\) to follow a reference command \(r[n]\). The model includes a digital controller, actuator delay, motor-thrust dynamics, vertical-body dynamics, and an altitude sensor. The exercise is written as a complete exam-style problem.

نظام التحكم في ارتفاع الدرون يجب أن يجعل الارتفاع المقاس \(y[n]\) يتبع المرجع \(r[n]\). يتضمن النموذج متحكماً رقمياً، تأخر المحركات، ديناميكا الدفع، ديناميكا الجسم العمودية، وحساس الارتفاع.

Before solving: what each block means

  • Reference \(r[n]\): desired altitude command.
  • Error \(e[n]\): difference between desired and measured altitude.
  • Controller: computes the correction command.
  • Motors and body: convert command into vertical motion.
  • Sensor: measures altitude and closes the loop.
قبل الحل: المرجع هو الارتفاع المطلوب، الخطأ هو الفرق بين المرجع والارتفاع المقاس، المتحكم يعطي أمر التصحيح، المحركات والجسم يحولان الأمر إلى حركة عمودية، والحساس يغلق الحلقة.

Q1Draw the block diagram and write the closed-loop model equation

Draw the control structure and use the following closed-loop characteristic equation:

\[D(z)=z^5-2.65z^4+2.72z^3-1.35z^2+0.32z-0.03\]

The numerator is selected as \(N(z)=0.01z^5\) so that the final closed-loop gain for a unit-step reference is equal to one.

تم اختيار البسط \(N(z)=0.01z^5\) حتى يكون الكسب النهائي للحلقة المغلقة مساوياً للواحد عند الدخل الدرجوي.
D Block diagram — Drone altitude closed-loop control
The diagram shows a standard negative-feedback altitude loop: the measured altitude is subtracted from the desired altitude, and the controller acts on the error.

Q2Find the transfer function of the complete closed-loop system

\[F(z)=\frac{Y(z)}{R(z)}=\frac{N(z)}{D(z)}\]
\[F(z)= \frac{0.01z^5} {z^5-2.65z^4+2.72z^3-1.35z^2+0.32z-0.03}\]
Important distinction: stability is decided by \(D(z)\), while tracking accuracy is studied using \(F(1)\).
الاستقرار يحدده المقام، أما الدقة النهائية في تتبع المرجع فتدرس من خلال \(F(1)\).
