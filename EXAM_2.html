Exercise 1 — Electrical factory supply chain model 10 pts

A factory receives electrical power from a main feeder. The power passes through a rectifier, a DC bus, inverter drives, and finally motorized production machines. The discrete equivalent of the energy-flow correction model is:

\[y[n]-1.05y[n-1]+0.37y[n-2]-0.045y[n-3]=0.08x[n]+0.05x[n-1]+0.02x[n-2]\]

The input signal represents the normalized electrical supply request:

\[x[n]=3u[n]-1.5(0.4)^nu[n]+2\delta[n-2]\]
Q1. Draw the block diagram and identify \(X(z)\), \(Y(z)\), storage/memory effects, and output. 2 pts
Q2. Compute \(X(z)\). 2 pts
Q3. Find \(H(z)=Y(z)/X(z)\). 2 pts
Q4. For \(x[n]=1\), derive the recurrence equation and compute \(y[n]\) up to \(n=10\). 2 pts
Q5. Decide stability using the characteristic equation and Jury. 2 pts

Exercise 2 — Fifth-order drone control system 10 pts

A drone altitude-control loop is represented by a fifth-order closed-loop model:

\[F(z)=\frac{Y(z)}{R(z)}=\frac{0.012z^5}{z^5-2.50z^4+2.41z^3-1.115z^2+0.245z-0.020}\]
Q1. Draw the detailed block diagram of the drone control loop. 2 pts
Q2. Extract the recurrence equation between \(r[n]\) and \(y[n]\). 2 pts
Q3. For \(r[n]=1\), compute \(y[n]\) and \(e[n]=1-y[n]\) up to \(n=10\). 2 pts
Q4. Compute the steady-state error using \(F(1)\). 2 pts
Q5. Verify stability using a full Jury table until the final reduced stage. 2 pts

Solution Method Guide

Signal/system exercise

  1. Draw the physical block diagram.
  2. Compute \(X(z)\) using linearity.
  3. Transform the difference equation.
  4. Find \(H(z)=Y(z)/X(z)\).
  5. Derive the recurrence equation.
  6. Compute values for \(n=0,\ldots,10\).
  7. Study stability from \(D(z)\).

Fifth-order control system

  1. Start from \(F(z)=N(z)/D(z)\).
  2. Use \(D(z)Y(z)=N(z)R(z)\).
  3. Divide by \(z^5\).
  4. Apply inverse Z-transform.
  5. Compute output values recursively.
  6. Compute \(F(1)\).
  7. Use Jury to confirm stability.

Detailed Solutions

Exercise 1 — Electrical factory supply chain model

Expanded solution plan. The solution follows five stages: first, interpret the factory blocks physically; second, compute the Z-transform of the supply request; third, derive the transfer function; fourth, compute and draw \(x[n]\) and \(y[n]\); finally, verify stability using the full Jury procedure.

Why this model is a supply-chain electrical model

The factory electrical chain behaves like a discrete dynamic system because power demand is not transmitted instantly to the machines. The rectifier, DC bus, inverter drives, and machine loads introduce memory. In the difference equation, memory appears through previous outputs \(y[n-1]\), \(y[n-2]\), and \(y[n-3]\). The delayed input terms \(x[n-1]\) and \(x[n-2]\) represent delayed influence of the supply request.

  1. Current input effect: \(0.08x[n]\) represents the immediate response to a new electrical request.
  2. Delayed input effect: \(0.05x[n-1]+0.02x[n-2]\) represents energy already injected into intermediate stages.
  3. Storage effect: \(1.05y[n-1]-0.37y[n-2]+0.045y[n-3]\) models the stored energy and correction dynamics.
  4. Stability question: if stored energy does not decay, the supply chain may oscillate or diverge; this is why Jury is required.

Exercise 2 — Fifth-order drone control system

Expanded solution plan. The fifth-order model is solved by converting the closed-loop transfer function into a recurrence equation, computing the step response, drawing \(x[n]\) and \(y[n]\), checking the steady-state error, and then validating stability with Jury.

Why a fifth-order drone model is realistic

A drone altitude loop can easily become high-order because several dynamics are combined: controller memory, motor delay, propeller thrust buildup, body vertical acceleration, and sensor feedback. The denominator has degree five, which means the current output depends on five previous output samples.

  1. Reference \(r[n]\): desired altitude command.
  2. Output \(y[n]\): measured altitude response.
  3. Fifth-order memory: the model uses \(y[n-1]\) through \(y[n-5]\).
  4. Stability requirement: all five closed-loop poles must be inside the unit circle.
2 Block diagram — fifth-order drone altitude control

Q1 — Structure

The reference altitude is compared with the measured altitude. The error enters a digital controller. The command passes through motor dynamics and the drone body, then the altitude sensor closes the loop.

Final Review

One-minute method

  1. Draw the block diagram first.
  2. Write the Z-domain equation.
  3. Find \(H(z)\) or \(F(z)\).
  4. Derive the recurrence equation.
  5. Compute the table up to \(n=10\) when requested.
  6. Draw the response curve \(y[n]\) from the computed table and interpret its shape.
  7. Use \(F(1)\) for steady-state error.
  8. Use Jury for stability from \(D(z)\).
  9. Draw \(y[n]\) from the table: horizontal axis \(n\), vertical axis output \(y[n]\), then interpret rise, overshoot, and final error.
