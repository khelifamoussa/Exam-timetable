Learning Map خريطة التعلم

Main idea. The Z-transform changes sequence operations into algebraic operations. This is why delays, convolution, and difference equations become easier to solve in the Z-domain.
الفكرة الأساسية. تحويل Z يحوّل العمليات على المتتاليات إلى عمليات جبرية، لذلك تصبح الإزاحة والالتفاف والمعادلات الفرقية أسهل في مجال Z.
\[x[n]\;\xleftrightarrow{\mathcal Z}\;X(z),\qquad y[n]=x[n]*h[n]\;\Longleftrightarrow\;Y(z)=X(z)H(z)\]

Main Properties الخواص الأساسية

Linearity
\[\mathcal Z\{a x_1[n]+b x_2[n]\}=aX_1(z)+bX_2(z)\]
Time Shifting
\[\mathcal Z\{x[n-k]\}=z^{-k}X(z)\]
Exponential Multiplication
\[\mathcal Z\{a^n x[n]\}=X\!\left(\frac{z}{a}\right)\]
Convolution
\[x[n]*h[n]\;\xleftrightarrow{\mathcal Z}\;X(z)H(z)\]
Transfer Function
\[H(z)=\frac{Y(z)}{X(z)},\qquad Y(z)=H(z)X(z)\]

Inverse Z-Transform Methods طرق التحويل العكسي

Use this when the expression already matches a standard pair.

\[\mathcal Z^{-1}\left\{\frac{1}{1-a z^{-1}}\right\}=a^n u[n]\]

Expand the expression as powers of \(z^{-1}\), then read the coefficients as sequence samples.

\[X(z)=\sum_{n=0}^{\infty}x[n]z^{-n}\]

Factor the denominator, write a sum of simple fractions, solve the constants, then apply standard inverse pairs.

\[\frac{1}{(1-z^{-1})(1-0.5z^{-1})}=\frac{A}{1-z^{-1}}+\frac{B}{1-0.5z^{-1}}\]

Applications التطبيقات

Difference equations

A linear difference equation becomes an algebraic equation in \(Y(z)\). With zero initial conditions:

\[y[n]-a y[n-1]=x[n]\quad\Longrightarrow\quad Y(z)(1-a z^{-1})=X(z)\]

System output

For an LTI system, multiply the input transform by the transfer function, then invert:

\[Y(z)=H(z)X(z)\quad\Longrightarrow\quad y[n]=\mathcal Z^{-1}\{Y(z)\}\]
