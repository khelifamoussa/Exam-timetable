Standard Z-Transform Pairs الأزواج القياسية لتحويل Z

Study Orientation / توجيه دراسي: Work from definitions to properties, then from properties to applications. Always state the ROC when the sequence has infinite duration.

These pairs are the building blocks. Most problems are solved by rewriting a signal or a rational expression until it matches one of these forms.

هذه الأزواج هي الأساس. أغلب المسائل تحل بإعادة كتابة الإشارة أو التعبير الجبري حتى يصبح مشابها لأحد هذه الأشكال.

Main Properties of the Z-Transform الخواص الرئيسية لتحويل Z

Linearity / الخطية

\(Z\{a x_1[n]+b x_2[n]\}=aX_1(z)+bX_2(z)\)

Break a complicated signal into simple parts, transform each part, then add the results.

نفكك الإشارة المعقدة إلى حدود بسيطة، نحول كل حد، ثم نجمع النتائج.

Time Shifting / الإزاحة الزمنية

\(Z\{x[n-k]\}=z^{-k}X(z)\)

A delay of \(k\) samples adds the factor \(z^{-k}\).

التأخير بمقدار \(k\) عينات يضيف العامل \(z^{-k}\).

Exponential Multiplication / الضرب في متتالية أسية

\(Z\{a^n x[n]\}=X\left(\frac{z}{a}\right)\)

Multiplying the time signal by \(a^n\) changes the argument of the transform.

ضرب الإشارة في \(a^n\) يغير وسيط التحويل في مجال Z.

Convolution / الالتفاف

\(y[n]=x[n]*h[n] \Rightarrow Y(z)=X(z)H(z)\)

A difficult time-domain summation becomes simple multiplication in the Z-domain.

مجموع الالتفاف الصعب في الزمن يتحول إلى ضرب بسيط في مجال Z.

Transfer Function / دالة التحويل: \(H(z)=\frac{Y(z)}{X(z)}\). Therefore, \(Y(z)=H(z)X(z)\). هذا هو الأساس الرياضي لتحليل الأنظمة الخطية الثابتة والمرشحات الرقمية.

Composite Signal Method منهجية التعامل مع الإشارات المركبة

Systematic Steps / الخطوات المنظمة

Identify components / تحديد المكونات: Separate \(u[n]\), \(a^n u[n]\), impulses, and delayed terms.
Use standard pairs / استعمال الأزواج القياسية: Transform each component alone.
Apply linearity / تطبيق الخطية: Add all transformed terms with their constants.
Find ROC / تحديد منطقة التقارب: Intersect the ROCs of all infinite-duration terms.

Worked Example / مثال محلول

Find the Z-transform of:

\(x[n]=2u[n]+(0.5)^n u[n]+\delta[n-1]\)

Step 1: Transform each term.

\(Z\{2u[n]\}=\frac{2}{1-z^{-1}},\quad Z\{(0.5)^n u[n]\}=\frac{1}{1-0.5z^{-1}},\quad Z\{\delta[n-1]\}=z^{-1}\)

Step 2: Combine.

\(X(z)=\frac{2}{1-z^{-1}}+\frac{1}{1-0.5z^{-1}}+z^{-1}\)

Step 3: ROC: \(|z|>1\), because it is the stricter condition.

الشرح: النبضة لا تقيد منطقة التقارب لأنها محدودة المدة، لذلك نأخذ تقاطع \(|z|>1\) و \(|z|>0.5\).

Inverse Z-Transform التحويل العكسي Z

Definition / التعريف

The inverse Z-transform converts \(X(z)\) back to the time-domain sequence \(x[n]\).

التحويل العكسي يحول التعبير \(X(z)\) من مجال Z إلى المتتالية الزمنية \(x[n]\).

\(x[n]=Z^{-1}\{X(z)\}\)

Why It Matters / لماذا هو مهم؟

Engineering interpretation needs the actual output sequence, not only the symbolic expression.

في التطبيقات الهندسية نحتاج إلى الخرج في الزمن حتى نفهم الاستجابة العابرة والمستقرة للنظام.

Partial Fraction Decomposition تفكيك الكسور الجزئية

Reliable Procedure / طريقة مضمونة

Factor the denominator / حلل المقام إلى عوامل.
Assume a simple fraction for every factor / افترض كسرا بسيطا لكل عامل.
Multiply by the common denominator / اضرب في المقام المشترك.
Expand and compare coefficients / وسع وقارن المعاملات.
Apply inverse standard pairs / طبق الأزواج العكسية القياسية.

Detailed Example / مثال مفصل

\(X(z)=\frac{1}{(1-z^{-1})(1-0.5z^{-1})}\)

1) Assume:

\(X(z)=\frac{A}{1-z^{-1}}+\frac{B}{1-0.5z^{-1}}\)

2) Clear denominators:

\(1=A(1-0.5z^{-1})+B(1-z^{-1})\)

3) Expand:

\(1=(A+B)+(-0.5A-B)z^{-1}\)

4) Compare coefficients:

\(A+B=1,\qquad -0.5A-B=0\)

5) Solve: \(B=-0.5A\), so \(A-0.5A=1\Rightarrow A=2\), and \(B=-1\).

6) Invert:

\(x[n]=2u[n]-(0.5)^n u[n]\)

الفكرة: كل حد أصبح مطابقا لزوج قياسي، لذلك صار التحويل العكسي مباشرا.

Engineering Applications تطبيقات هندسية

Difference Equation / المعادلة الفرقية

Solve with zero initial conditions:

\(y[n]-0.5y[n-1]=u[n]\)

Take Z-transform:

\(Y(z)-0.5z^{-1}Y(z)=\frac{1}{1-z^{-1}}\)

Factor:

\(Y(z)(1-0.5z^{-1})=\frac{1}{1-z^{-1}}\)

So:

\(Y(z)=\frac{1}{(1-z^{-1})(1-0.5z^{-1})}\)

From partial fractions:

\(y[n]=2u[n]-(0.5)^n u[n]\)

System Output / خرج النظام

Given:

\(H(z)=\frac{1}{1-0.4z^{-1}},\qquad x[n]=u[n]\)

Input transform:

\(X(z)=\frac{1}{1-z^{-1}}\)

Output transform:

\(Y(z)=\frac{1}{(1-z^{-1})(1-0.4z^{-1})}\)

Let \(Y(z)=\frac{A}{1-z^{-1}}+\frac{B}{1-0.4z^{-1}}\). Then \(A+B=1\), \(-0.4A-B=0\). Hence \(A=\frac{5}{3}\), \(B=-\frac{2}{3}\).

\(y[n]=\frac{5}{3}u[n]-\frac{2}{3}(0.4)^n u[n]\)

Detailed Solved Exercises تمارين محلولة بالتفصيل

Idea / الفكرة: Use linearity because the signal is a sum.

\(Z\{\delta[n]\}=1\), and \(Z\{u[n]\}=\frac{1}{1-z^{-1}}\).

\(X(z)=1+\frac{1}{1-z^{-1}},\qquad |z|>1\)

النبضة لا تقيد منطقة التقارب، لذلك ROC تأتي من \(u[n]\).

Start from \(Z\{u[n]\}=\frac{1}{1-z^{-1}}\). A delay by 3 samples gives multiplication by \(z^{-3}\).

\(X(z)=\frac{z^{-3}}{1-z^{-1}},\qquad |z|>1\)

هذا يعني أن الخطوة تبدأ عند \(n=3\) بدل \(n=0\).

Use \(Z\{a^n u[n]\}=\frac{1}{1-a z^{-1}}\). Here \(a=0.5\), then multiply by 2.

\(X(z)=\frac{2}{1-0.5z^{-1}},\qquad |z|>0.5\)

Separate terms:

\(Z\{3u[n]\}=\frac{3}{1-z^{-1}},\qquad Z\{2\delta[n-2]\}=2z^{-2}\)

Combine:

\(X(z)=\frac{3}{1-z^{-1}}+2z^{-2},\qquad |z|>1\)

Transform each component:

\(X(z)=\frac{1}{1-z^{-1}}+\frac{1}{1-0.3z^{-1}}\)

ROC is the intersection: \(|z|>1\) and \(|z|>0.3\), so:

\(|z|>1\)

Convolution in time becomes multiplication in Z-domain:

\(Y(z)=X(z)H(z)\)

هذا هو السبب في أن تحويل Z مهم جدا في الأنظمة الخطية الثابتة.

Compare with \(\frac{1}{1-a z^{-1}}\leftrightarrow a^n u[n]\). Here \(a=0.2\).

\(x[n]=(0.2)^n u[n]\)

Since \(\frac{1}{1-z^{-1}}\leftrightarrow u[n]\), multiply by 2:

\(x[n]=2u[n]\)

Use the delayed impulse pair:

\(Z^{-1}\{z^{-k}\}=\delta[n-k]\)

For \(k=3\):

\(x[n]=\delta[n-3]\)

Invert term by term:

\(Z^{-1}\{3z^{-2}\}=3\delta[n-2],\quad Z^{-1}\{2z^{-4}\}=2\delta[n-4]\)
\(x[n]=3\delta[n-2]+2\delta[n-4]\)

Assume:

\(X(z)=\frac{A}{1-z^{-1}}+\frac{B}{1-0.3z^{-1}}\)

Clear denominators:

\(1=A(1-0.3z^{-1})+B(1-z^{-1})\)

Compare:

\(A+B=1,\quad -0.3A-B=0\)

From the second equation, \(B=-0.3A\). Substitute: \(0.7A=1\Rightarrow A=\frac{10}{7}\), and \(B=-\frac{3}{7}\).

\(x[n]=\frac{10}{7}u[n]-\frac{3}{7}(0.3)^n u[n]\)

Each term already matches a standard inverse pair:

\(x[n]=3(0.7)^n u[n]+2u[n]\)

With zero initial conditions:

\(Y(z)-0.2z^{-1}Y(z)=\frac{1}{1-z^{-1}}\)
\(Y(z)=\frac{1}{(1-z^{-1})(1-0.2z^{-1})}\)

Assume \(\frac{A}{1-z^{-1}}+\frac{B}{1-0.2z^{-1}}\). Then:

\(A+B=1,\quad -0.2A-B=0\)

So \(A=1.25\), \(B=-0.25\).

\(y[n]=1.25u[n]-0.25(0.2)^n u[n]\)

Input transform:

\(X(z)=\frac{1}{1-0.5z^{-1}}\)

Equation transform:

\(Y(z)(1-0.3z^{-1})=\frac{1}{1-0.5z^{-1}}\)
\(Y(z)=\frac{1}{(1-0.3z^{-1})(1-0.5z^{-1})}\)

Assume \(\frac{A}{1-0.3z^{-1}}+\frac{B}{1-0.5z^{-1}}\).

\(1=A(1-0.5z^{-1})+B(1-0.3z^{-1})\)

Compare: \(A+B=1\), \(-0.5A-0.3B=0\). Solving gives \(A=-1.5\), \(B=2.5\).

\(y[n]=-1.5(0.3)^n u[n]+2.5(0.5)^n u[n]\)

Input transform:

\(X(z)=\frac{1}{1-z^{-1}}\)

Output transform:

\(Y(z)=H(z)X(z)=\frac{1}{(1-z^{-1})(1-0.5z^{-1})}\)

From the central example:

\(y[n]=2u[n]-(0.5)^n u[n]\)

الجزء \(2u[n]\) هو الحالة المستقرة، والجزء \(-(0.5)^n u[n]\) هو الجزء العابر الذي يختفي مع الزمن.

Compute:

\(Y(z)=\frac{1}{(1-z^{-1})(1-0.4z^{-1})}\)

Assume \(\frac{A}{1-z^{-1}}+\frac{B}{1-0.4z^{-1}}\).

\(A+B=1,\quad -0.4A-B=0\)

Then \(A=\frac{5}{3}\), \(B=-\frac{2}{3}\).

\(y[n]=\frac{5}{3}u[n]-\frac{2}{3}(0.4)^n u[n]\)

Common Mistakes أخطاء شائعة

Forgetting \(z^{-k}\) / نسيان عامل التأخير: For \(x[n-k]\), always multiply by \(z^{-k}\).
Ignoring ROC / تجاهل منطقة التقارب: ROC is required for infinite-duration sequences.
Wrong partial fraction form / صيغة خاطئة للكسور: Every denominator factor needs its own constant.
Forgetting \(u[n]\) / نسيان الخطوة: Causal exponentials must be written as \(a^n u[n]\).
