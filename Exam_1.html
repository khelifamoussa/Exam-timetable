Concepts Before Solving

1. Important correction: stability is decided by the denominator

Sometimes students say: “we solve the numerator to decide stability.” This is not correct for transfer-function stability. Stability is decided from the poles, which are roots of the denominator.

\[F(z)=\frac{N(z)}{D(z)}\]
\[\text{Poles: }D(z)=0,\qquad \text{Zeros: }N(z)=0\]
Stable if all roots of \(D(z)\) satisfy \(|z_i|<1\).
If one root has \(|z_i|=1\) or \(|z_i|>1\), the discrete-time system is not asymptotically stable.
تصحيح مهم: الاستقرار يُحكم من جذور المقام، وليس البسط. جذور المقام هي الأقطاب، وهي التي تحدد الاستقرار.

2. Delay rule for inverse Z-transform

The main rule used to convert a Z-domain relation into a recurrence equation is:

\[z^{-1}Y(z)\longleftrightarrow y[n-1]\]
\[z^{-2}Y(z)\longleftrightarrow y[n-2]\]

So when a denominator is second-order, the recurrence usually contains \(y[n-1]\) and \(y[n-2]\).

عندما نقسم على أعلى قوة من \(z\)، تظهر حدود مثل \(z^{-1}\) و \(z^{-2}\)، وهي تمثل التأخر: \(y[n-1]\) و \(y[n-2]\).

3. Jury method for degree 3 and above

For degree 1 or 2, roots or simple second-order Jury conditions are easy. For degree 3 and above, the Jury table becomes very useful.

\[D(z)=a_0z^n+a_1z^{n-1}+\cdots+a_n\]

Preliminary conditions:

\[D(1)>0,\qquad (-1)^nD(-1)>0,\qquad |a_n|

Reduction formula:

\[q_k=p_0p_k-p_mp_{m-k}\]

1 Closed-loop first-order motor without PI

Q1 — Closed-loop transfer function without PI

For unity negative feedback, the closed-loop transfer function is:

\[F_0(z)=\frac{G(z)}{1+G(z)}\]

Substitute the motor model:

\[F_0(z)=\frac{\frac{0.4z}{z-0.6}}{1+\frac{0.4z}{z-0.6}}\]

To simplify, write the denominator over the same denominator \(z-0.6\):

\[1+\frac{0.4z}{z-0.6}=\frac{z-0.6}{z-0.6}+\frac{0.4z}{z-0.6}\]
\[1+\frac{0.4z}{z-0.6}=\frac{z-0.6+0.4z}{z-0.6}=\frac{1.4z-0.6}{z-0.6}\]

Now divide two fractions:

\[F_0(z)=\frac{0.4z}{z-0.6}\cdot\frac{z-0.6}{1.4z-0.6}\]
\[F_0(z)=\frac{0.4z}{1.4z-0.6}\]
استعملنا قانون الحلقة المغلقة، ثم وحدنا المقامات، وبعدها اختصرنا \(z-0.6\).

Q2 — Relation between input and output

By definition:

\[F_0(z)=\frac{Y(z)}{X(z)}=\frac{0.4z}{1.4z-0.6}\]

Cross-multiply:

\[(1.4z-0.6)Y(z)=0.4zX(z)\]

Divide by \(z\), because we want powers \(z^{-1}\):

\[(1.4-0.6z^{-1})Y(z)=0.4X(z)\]

Apply inverse Z-transform:

\[1.4y[n]-0.6y[n-1]=0.4x[n]\]

Solve for \(y[n]\):

\[y[n]=\frac{0.6}{1.4}y[n-1]+\frac{0.4}{1.4}x[n]\]
\[y[n]=0.4286y[n-1]+0.2857x[n]\]

Q3 — Stability without PI

The denominator is:

\[D_0(z)=1.4z-0.6\]

Set it equal to zero:

\[1.4z-0.6=0\]
\[1.4z=0.6\]
\[z=\frac{0.6}{1.4}=0.4286\]

Now compare with the unit-circle radius:

\[|z|=0.4286<1\]
The closed-loop motor without PI is stable.
Again: we solved the denominator. The numerator \(0.4z\) gives a zero, not the stability pole.

Q4 — New closed-loop transfer function with PI

The controller is:

\[C(z)=K_p+\frac{K_iz}{z-1}\]

Put it over a common denominator:

\[C(z)=\frac{K_p(z-1)+K_iz}{z-1}\]

Substitute \(K_p=0.8\), \(K_i=0.2\):

\[C(z)=\frac{0.8(z-1)+0.2z}{z-1}\]
\[C(z)=\frac{0.8z-0.8+0.2z}{z-1}=\frac{z-0.8}{z-1}\]

The open-loop transfer function is:

\[L(z)=C(z)G(z)=\frac{z-0.8}{z-1}\cdot\frac{0.4z}{z-0.6}\]
\[L(z)=\frac{0.4z(z-0.8)}{(z-1)(z-0.6)}\]

The closed-loop function is:

\[F_{PI}(z)=\frac{L(z)}{1+L(z)}\]
\[F_{PI}(z)=\frac{0.4z(z-0.8)}{(z-1)(z-0.6)+0.4z(z-0.8)}\]

Expand both parts of the denominator:

\[(z-1)(z-0.6)=z^2-1.6z+0.6\]
\[0.4z(z-0.8)=0.4z^2-0.32z\]

Add them:

\[D_{PI}(z)=z^2-1.6z+0.6+0.4z^2-0.32z\]
\[D_{PI}(z)=1.4z^2-1.92z+0.6\]

The numerator is:

\[N_{PI}(z)=0.4z^2-0.32z\]

Therefore:

\[F_{PI}(z)=\frac{0.4z^2-0.32z}{1.4z^2-1.92z+0.6}\]

Stability with PI

Use the denominator:

\[D_{PI}(z)=1.4z^2-1.92z+0.6\]

For a second-order polynomial, we can solve the roots or use second-order Jury. Here, use the roots:

\[z=\frac{1.92\pm\sqrt{(-1.92)^2-4(1.4)(0.6)}}{2(1.4)}\]
\[z=\frac{1.92\pm\sqrt{3.6864-3.36}}{2.8}\]
\[z=\frac{1.92\pm\sqrt{0.3264}}{2.8}\]
\[\sqrt{0.3264}=0.5713\]
\[z_1=\frac{1.92+0.5713}{2.8}=0.8898,\qquad z_2=\frac{1.92-0.5713}{2.8}=0.4817\]
\[|z_1|=0.8898<1,\qquad |z_2|=0.4817<1\]
The system with PI is stable.

Q5 — Relation between input and output and inverse Z-transform

Start from:

\[\frac{Y(z)}{X(z)}=\frac{0.4z^2-0.32z}{1.4z^2-1.92z+0.6}\]

Cross-multiply:

\[(1.4z^2-1.92z+0.6)Y(z)=(0.4z^2-0.32z)X(z)\]

Divide by \(z^2\). This is the key step because it creates \(z^{-1}\) and \(z^{-2}\):

\[(1.4-1.92z^{-1}+0.6z^{-2})Y(z)=(0.4-0.32z^{-1})X(z)\]

Now apply inverse Z-transform term by term:

\[1.4Y(z)\longleftrightarrow 1.4y[n]\]
\[-1.92z^{-1}Y(z)\longleftrightarrow -1.92y[n-1]\]
\[0.6z^{-2}Y(z)\longleftrightarrow 0.6y[n-2]\]
\[0.4X(z)\longleftrightarrow 0.4x[n]\]
\[-0.32z^{-1}X(z)\longleftrightarrow -0.32x[n-1]\]

So the recurrence equation is:

\[1.4y[n]-1.92y[n-1]+0.6y[n-2]=0.4x[n]-0.32x[n-1]\]

Solve for \(y[n]\):

\[y[n]=\frac{1.92}{1.4}y[n-1]-\frac{0.6}{1.4}y[n-2]+\frac{0.4}{1.4}x[n]-\frac{0.32}{1.4}x[n-1]\]
\[y[n]=1.3714y[n-1]-0.4286y[n-2]+0.2857x[n]-0.2286x[n-1]\]

Q6 — Reference recurrence equation, unit-step input, and discrete plot of \(Y[n]\)

This question must be solved in the same logical order as Exercise 1: first derive the reference relation between input and output, then choose the input \(X[n]\), then compute the samples, then draw \(Y[n]\) as separated discrete points.

Step 1 — Start from the Z-domain input-output relation

From the closed-loop transfer function found previously:

\[ \frac{Y(z)}{X(z)}= \frac{0.1275z^4-0.105z^3} {1.1275z^4-2.205z^3+1.49z^2-0.435z+0.045} \]

Cross-multiply to get the relation between \(X(z)\) and \(Y(z)\):

\[ (1.1275z^4-2.205z^3+1.49z^2-0.435z+0.045)Y(z) =(0.1275z^4-0.105z^3)X(z) \]

Step 2 — Convert the relation into delayed powers

Because the highest power is \(z^4\), divide the full equation by \(z^4\). This is the key step that creates delays:

\[ (1.1275-2.205z^{-1}+1.49z^{-2}-0.435z^{-3}+0.045z^{-4})Y(z) =(0.1275-0.105z^{-1})X(z) \]
The rule is: \[ z^{-1}Y(z)\leftrightarrow y[n-1],\quad z^{-2}Y(z)\leftrightarrow y[n-2],\quad z^{-3}Y(z)\leftrightarrow y[n-3],\quad z^{-4}Y(z)\leftrightarrow y[n-4] \]

Step 3 — Apply inverse Z-transform

Transform each term separately:

\[ 1.1275Y(z)\leftrightarrow 1.1275y[n] \]
\[ -2.205z^{-1}Y(z)\leftrightarrow -2.205y[n-1] \]
\[ 1.49z^{-2}Y(z)\leftrightarrow 1.49y[n-2] \]
\[ -0.435z^{-3}Y(z)\leftrightarrow -0.435y[n-3] \]
\[ 0.045z^{-4}Y(z)\leftrightarrow 0.045y[n-4] \]

For the input terms:

\[ 0.1275X(z)\leftrightarrow 0.1275x[n],\qquad -0.105z^{-1}X(z)\leftrightarrow -0.105x[n-1] \]

So the reference recurrence equation is:

\[ 1.1275y[n]-2.205y[n-1]+1.49y[n-2]-0.435y[n-3]+0.045y[n-4] =0.1275x[n]-0.105x[n-1] \]

Step 4 — Solve for the current output \(y[n]\)

Move all previous-output terms to the right-hand side and divide by \(1.1275\):

\[ y[n]=1.956y[n-1]-1.322y[n-2]+0.386y[n-3]-0.0399y[n-4]+0.1131x[n]-0.0931x[n-1] \]

This is the formula used to generate the output samples.

هذه هي العلاقة المرجعية التي نستعملها لحساب \(Y[n]\) عينة بعد عينة عندما يكون الدخل \(X[n]\) إشارة وحدة.

Step 5 — Assume the input is the unit-step signal

\[ x[n]=1,\qquad n\ge 0 \]

Before the beginning of the sequence, use zero initial conditions:

\[ x[-1]=0,\qquad y[-1]=y[-2]=y[-3]=y[-4]=0 \]

Step 6 — Compute the first samples of \(Y[n]\)

At \(n=0\):

\[ y[0]=0.1131x[0]-0.0931x[-1]=0.1131(1)-0.0931(0)=0.1131 \]

At \(n=1\):

\[ y[1]=1.956y[0]+0.1131x[1]-0.0931x[0] \]
\[ y[1]=1.956(0.1131)+0.1131-0.0931=0.2413 \]

At \(n=2\):

\[ y[2]=1.956y[1]-1.322y[0]+0.1131-0.0931 \]
\[ y[2]=1.956(0.2413)-1.322(0.1131)+0.0200=0.3425 \]

2 Third-order plant with PI controller

Q2 — Open-loop transfer function

First simplify the PI controller:

\[C(z)=K_p+\frac{K_iz}{z-1}\]
\[C(z)=\frac{K_p(z-1)+K_iz}{z-1}\]
\[C(z)=\frac{(K_p+K_i)z-K_p}{z-1}\]

Substitute \(K_p=0.7\), \(K_i=0.15\):

\[C(z)=\frac{0.85z-0.7}{z-1}\]

The plant is:

\[G_2(z)=\frac{0.15z^3}{z^3-1.1z^2+0.39z-0.045}\]

Therefore:

\[L(z)=C(z)G_2(z)\]
\[L(z)=\frac{0.15z^3(0.85z-0.7)}{(z-1)(z^3-1.1z^2+0.39z-0.045)}\]

Expand the numerator:

\[0.15z^3(0.85z-0.7)=0.1275z^4-0.105z^3\]

Q3 — Closed-loop transfer function

For unity negative feedback:

\[F(z)=\frac{L(z)}{1+L(z)}\]

So:

\[F(z)=\frac{0.1275z^4-0.105z^3}{(z-1)(z^3-1.1z^2+0.39z-0.045)+0.1275z^4-0.105z^3}\]

Now expand the plant denominator multiplied by \((z-1)\):

\[(z-1)(z^3-1.1z^2+0.39z-0.045)\]
\[=z(z^3-1.1z^2+0.39z-0.045)-1(z^3-1.1z^2+0.39z-0.045)\]
\[=z^4-1.1z^3+0.39z^2-0.045z-z^3+1.1z^2-0.39z+0.045\]
\[=z^4-2.1z^3+1.49z^2-0.435z+0.045\]

Add the controller numerator term:

\[D(z)=z^4-2.1z^3+1.49z^2-0.435z+0.045+0.1275z^4-0.105z^3\]

Thus:

\[F(z)=\frac{0.1275z^4-0.105z^3}{1.1275z^4-2.205z^3+1.49z^2-0.435z+0.045}\]

Final Review

What students must remember

  1. For stability, solve the denominator \(D(z)=0\), not the numerator.
  2. If \(|z_i|<1\) for every pole, the system is stable.
  3. For PI controller:
    \[C(z)=K_p+\frac{K_iz}{z-1}=\frac{(K_p+K_i)z-K_p}{z-1}\]
  4. To obtain a recurrence equation, divide by the highest power of \(z\), then use:
    \[z^{-1}Y(z)\leftrightarrow y[n-1],\qquad z^{-2}Y(z)\leftrightarrow y[n-2]\]
  5. For degree 3 or higher, use Jury table carefully until the final second-order stage.
